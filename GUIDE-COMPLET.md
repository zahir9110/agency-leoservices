## ✨ Caractéristiques Principales

### 1. **Effet Pixelisé Dynamique** 🔲

Le site intègre un canvas HTML5 avec un effet visuel unique :

- **Particules pixelisées animées** en temps réel
- **Interaction souris** : les particules s'éloignent au passage du curseur
- **Connexions dynamiques** entre particules proches (lignes lumineuses)
- **Animation fluide** avec RequestAnimationFrame pour des performances optimales
- **Responsive** : s'adapte automatiquement à la taille de l'écran

**Personnalisation des particules :**
```javascript
let pixelSize = 20;  // Taille des pixels
const particleCount = Math.floor((canvas.width * canvas.height) / 15000);
// Modifier 15000 pour ajuster la densité
```

### 2. **Colorimétrie Adaptative Intelligente** 💡

Widget de contrôle en haut à droite avec 3 modes :

#### **Mode Auto** (par défaut)
- Détecte automatiquement les préférences système (dark/light mode)
- S'adapte à l'API Ambient Light Sensor si disponible
- Change automatiquement selon le contexte

#### **Mode Sombre**
- Fond noir profond (#000000)
- Vert fluo intense (#00ff41)
- Contraste maximal
- Opacité canvas à 60%
- Idéal pour environnements lumineux

#### **Mode Clair**
- Fond gris foncé (#0f0f0f)
- Vert modéré (#00cc33)
- Moins agressif visuellement
- Opacité canvas à 40%
- Idéal pour environnements sombres

## 🎛️ Personnalisation

### Modifier les Couleurs

Dans la section `:root` du CSS :

```css
:root {
    --neon-green: #00ff41;    /* Couleur principale */
    --dark-green: #00cc33;    /* Couleur secondaire */
    --deep-black: #000000;    /* Fond principal */
}
```

### Ajuster l'Effet Pixelisé

Dans le JavaScript :

```javascript
// Taille des pixels (plus grand = pixels plus visibles)
let pixelSize = 20;

// Densité des particules (nombre plus petit = plus de particules)
const particleCount = Math.floor((canvas.width * canvas.height) / 15000);
```

### Modifier le Contenu

Toutes les sections sont clairement identifiées dans le HTML avec des commentaires :

```html
<!-- Section Hero -->
<!-- Section Stats -->
<!-- Section Benefits -->
<!-- etc. -->
```

### Personnaliser les Animations

Vitesses d'animation dans le CSS :

```css
.scroll-reveal {
    transition: all 0.8s ease;  /* Modifier la durée */
}

@keyframes fadeInUp {
    /* Personnaliser l'animation */
}
```
